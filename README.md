# Computer Lab Management System

## Developed By

Fahad Mahar
Sabreena Asad

## Project Description

Computer Lab Management System is a C++ based application developed using Object-Oriented Programming concepts and File Handling.

## Features

* Login System
* Admin Management
* Student Management
* PC Management
* Update/Delete/Search PCs
* Issue Reporting
* Dashboard
* Booking System
* Session Management

## OOP Concepts

* Encapsulation
* Inheritance
* Polymorphism
* Abstraction

## Technologies

* C++
* Dev-C++
* File Handling
